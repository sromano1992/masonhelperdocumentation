Detailed Description

This visitor will get information about variables constructor. Implicit parameter is the compilation unit of class that contains the variable. Explicit parameter is the parameter that we search in compilation Unit. ES. simStateCU.accept(new FieldInitializerVisitor(myParameter)) will search all parameters in simStateCU and when myParameter is found will extract a list of constructor arguments.
